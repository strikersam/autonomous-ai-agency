# Autonomous Agency Setup
## Introduction
This guide provides an overview of the autonomous agency setup for the local-llm-server.
## Configuration
To configure the autonomous agency, follow these steps:
1. Create a new GitHub secret for ANTHROPIC_API_KEY.
2. Create a new GitHub secret for GH_PAT.
## Workflows
The following workflows are enabled for autonomous operation:
* agency-cycle.yml
* continuous-improvement.yml
* weekly-trend-digest.yml
* ci-failure-autofix.yml
* perplexity-maintenance.yml
