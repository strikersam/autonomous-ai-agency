# Agent Orchestration with Claude

## Setup
1. Replace `YOUR_CLAUDE_API_KEY` in `config.example.json`
2. Run `python orchestrator/agent_orchestrator.py deploy --config config.json`

## Usage
### Deploy Agent
`python orchestrator/agent_orchestrator.py deploy --agent <agent_name>`
### List Agents
`python orchestrator/agent_orchestrator.py list`
