# Twitter Insights — Issue #228

> Status: Documented | Date: 2026-06-01

## Summary

This document captures insights and discussion points from Issue #228 concerning
social media presence optimization and community growth strategies.

## Key Observations

1. **Discoverability**: Projects with consistent social presence see 3-5x more
   GitHub stars in the first 6 months compared to those relying on organic search alone.

2. **Developer Advocates**: The most successful open-source AI projects have
   dedicated developer advocates who engage with the community daily.

3. **Content Cadence**: A minimum of 2-3 technical posts per week maintains
   algorithmic visibility on developer-focused social platforms.

4. **Cross-Promotion**: Linking between GitHub Discussions, Twitter/X threads,
   and Discord communities creates a self-reinforcing engagement loop.
