# Managed Agents Dreams Skill

**Inspired by:** Claude's Managed Agents with Dreams consolidation

**Purpose:** Consolidate and replay learning from agent sessions for continuous improvement.

## Implementation

- **ManagedAgentDreams** — consolidate patterns from successful sessions
- **SessionMemory** — record decisions, outcomes, errors per session
- **Dream** — learned patterns with success metrics
- **Pattern replay** — provide agents with proven patterns
