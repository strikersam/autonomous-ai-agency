Frequently Asked Questions

Everything you need to know to get started with Agency Core v5.

What is a Company Graph?
The Company Graph is the canonical metadata backbone of Agency Core v5. It acts as the single source of truth of your systems (Shopify, Contentful, repositories), environment endpoints, credentials, connected docs, prioritized goals, and provisioned specialist playbooks. Instead of stateless prompting, agents read and write directly to this structured graph.
How does website technological scanning work?
When you trigger a scan, our active scanning engine performs an async HTTP crawl. It searches for specific frameworks, third-party libraries, tracking pixels, meta tags, and JS bundle signatures. It matches signatures to over 150 business systems to infer confidence levels and map dependencies.
How does human-in-the-loop (HITL) approval work?
Safety is a priority. For tasks requiring sensitive changes (e.g. updating production code, merging PRs, changing prices, modifying CRM settings), the workflow engine automatically suspends progress, saves a checkpoint state, and triggers an approval alert on the chat control plane, requesting explicit human verification.
Is my source code and API key secure?
Yes. All secrets and access tokens are stored securely in local database schemas using symmetric AES encryption. Furthermore, execution tasks are isolated inside separate sandboxed container runtimes, ensuring that agents can never corrupt host system directories.
