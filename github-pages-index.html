FAQ

Common questions about Autonomous AI Agency.

What does "autonomous AI agency" actually mean?

It means the platform picks up tasks (from chat, quick-notes, scheduled jobs, or issue ingestion), classifies them, plans execution, selects the right specialist agents, runs them through a typed workflow with verification and judging, and persists evidence into the Company Graph — all without a human having to manually trigger each step. You still approve risky work via the HITL gate.

Is this a ChatGPT wrapper?

No. Autonomous AI Agency routes to your own LLM providers — local Ollama models, cloud APIs (DeepSeek, OpenRouter, Anthropic, Together, etc.), or any OpenAI-compatible endpoint. No dependency on a single vendor.

How does the website scan work?

The scanner performs a multi-pass inspection: HTTP headers, HTML metadata, JavaScript bundle analysis, Wappalyzer-style fingerprinting, and structured AI extraction. It detects CMS, PIM, OMS, DAM, analytics, commerce, CDN, and framework stacks.

What are "specialists"?

Specialists are agent profiles auto-provisioned based on what the scanner detects. Engineering specialists cover backend, frontend, security, testing, and DevOps. Business specialists cover SEO, analytics, ecommerce, content strategy, CRM, and domain-specific workflows. Each specialist has routing rules, skill bindings, and typed inputs/outputs.

What skills are included?

28 skills total: 7 production-ready (ECC cross-harness orchestration, Obsidian Knowledge Graph, Workflow Engine, Memory Consolidation, Fabric Patterns, Financial Analyst, Agentic Agile) and 19 gated behind a feature flag (council-review, implementation-planner, test-first-executor, Graphify, and others). All skills are typed, tested, and bindable to specialists.

Is this production-ready?

The core path is stable: company onboarding, Company Graph, specialist provisioning, direct chat, workflow engine, doctor/diagnostics, and the skills core. Features marked "beta" or "experimental" (async agent jobs, multi-agent swarm, peer sync, agentic portfolio management) are gated behind feature flags and should not be relied on in production without testing.
